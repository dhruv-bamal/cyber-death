## в—Ҳ Overview

> *The grid is dark. The AI is awake. The signal is yours to follow вҖ” or destroy.*

**CyberDeath 2.0** is the second installment in the CyberDeath series (Explore the first part in pinned repo) вҖ” a browser-based interactive fiction game set in a fractured cyber-future. It builds upon the original narrative while functioning as a fully **standalone experience**.

The game fuses branching storytelling with real-time interaction mechanics, placing you inside a world of rogue AI systems, encrypted signals, and impossible choices between human survival and machine evolution.

## в—Ҳ Gameplay Mechanics

The experience is built on two interlocking systems:

```
NARRATIVE LAYER                     ACTION LAYER
в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ                      в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
Decision Nodes                      Timed Input Events
      вҶ“                                   вҶ“
Branching Paths              вҶ’      Outcome Modifiers
      вҶ“                                   вҶ“
Story State Updates          вҶ’      World Consequence
```

- **Decision Nodes** вҖ” Player choices drive all narrative progression
- **Branching Paths** вҖ” Multiple storylines, multiple endings
- **Interactive Events** вҖ” Certain scenes require direct user input
- **Timed Action System** вҖ” Speed of response changes what happens next
